Detailed Description

template<PositionProvider PositionSource, InstancableType... InstanceData>
class glhelp::InstancedMesh3d< PositionSource, InstanceData >

Allows for creation of instanced 3D meshes. Each instance can have its own set of data provided as instance attributes. The attributes will be available in the shader as per-instance vertex attributes, first one will be located just after Mesh3D parameters.
